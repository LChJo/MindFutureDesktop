# 插件市场使用指南

## 概述

插件市场是 MindFuture Desktop 的核心功能模块，用于管理 Claude Code 的插件生态系统。它提供了完整的插件发现、安装、管理和技能/代理浏览功能。

---

## 主要功能模块

### 1. 市场管理（市场 Tab）

#### 插件源管理
- 支持添加多个插件市场源（Marketplace）
- 内置官方市场源（如 claude-plugins-official、mf-marketplace）
- 可自定义添加第三方市场源
- 支持市场源的更新和移除

#### 插件浏览与安装
- 按市场源分类浏览所有可用插件
- 显示插件名称、描述、包含的技能列表
- 一键安装/卸载插件
- 支持启用/禁用已安装的插件

#### 插件状态指示
- 清晰显示插件安装状态（已安装/未安装）
- 显示插件启用状态（已启用/已禁用）
- 实时显示操作进度（安装中/卸载中）

#### 维护工具
- 插件注册表修复功能
- 解决安装状态不一致问题

---

### 2. Skill 管理（Skill 管理 Tab）

#### 本地技能
- 浏览用户目录下的本地技能（`~/.claude/commands/`）
- 显示技能名称、标题、描述
- 快速打开技能文件目录
- 创建桌面快捷方式

#### 插件技能
- 自动扫描所有已安装插件中的技能
- 按市场源分类展示（子页签筛选）
- 显示技能所属的插件信息
- 一键跳转到对应插件查看详情

#### 技能展示
| 图标颜色 | 含义 |
|---------|------|
| 紫色 | 本地技能 |
| 绿色 | 插件技能 |

---

### 3. Agent 管理（Agent 管理 Tab）

#### 本地代理
- 浏览用户目录下的本地代理（`~/.claude/agents/`）
- 显示代理名称、标题、描述
- 快速打开代理文件目录
- 创建桌面快捷方式

#### 插件代理
- 自动扫描所有已安装插件中的代理
- 按市场源分类展示（子页签筛选）
- 显示代理所属的插件信息
- 一键跳转到对应插件查看详情

#### 代理展示
| 图标颜色 | 含义 |
|---------|------|
| 蓝色 | 本地代理 |
| 橙色 | 插件代理 |

---

## 技术特点

| 特性 | 说明 |
|------|------|
| 实时刷新 | 支持手动刷新插件列表和技能/代理列表 |
| 智能搜索 | "查看插件"按钮精准定位技能/代理所属的插件 |
| 分类筛选 | 子页签按来源（本地/各市场源）动态生成 |
| 状态同步 | 与 Claude Code 设置文件实时同步 |
| 跨平台 | 支持 Windows 路径规范 |

---

## 用户操作流程

```
1. 添加市场源
      ↓
2. 浏览插件
      ↓
3. 安装插件
      ↓
4. 启用插件
      ↓
5. 在 Skill/Agent 管理中查看已安装的技能和代理
      ↓
6. 创建快捷方式或直接在 Claude Code 中使用
```

| 类型 | 路径 |

| 本地代理 | `~/.claude/agents/` |
| 启用配置 | `~/.claude/settings.json` |

## 常见问题

### Q: 插件安装后不生效怎么办？
A: 请确保插件已启用（绿色状态），如果问题持续，可以尝试使用"修复注册表"功能。

### Q: 如何添加第三方插件源？
A: 在市场 Tab 中点击"添加市场源"，输入 GitHub 仓库地址或本地路径即可。

### Q: 技能/代理创建的快捷方式在哪里？
A: 快捷方式会显示在 MindFuture Desktop 的桌面上，双击即可快速调用对应的 Claude 技能或代理。
